Employee_attrition
This project analyzes the key factors behind employee attrition using the IBM HR dataset.   It includes data cleaning, EDA, visualizations, insights from data

Project Objective

To understand:
Why employees leave the company  
Which factors influence attrition the most  
Build ML models to predict employee attrition  

Dataset Information

Rows: 1470  
Columns: 33  
Target Variable: Employee Attrition

Exploratory Data Analysis (Key Insights)

Overall Attrition Rate 16%

Final Conclusion

The company is losing young, low-income, overworked employees at higher rates.  
Workload, salary, and early-career instability are major contributors.
